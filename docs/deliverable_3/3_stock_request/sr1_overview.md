# กระบวนการเบิกวัสดุคงคลัง (Stock Request)

ในระบบนี้การขอเบิกวัสดุหมายถึงการเบิกวัสดุสำนักงานเป็นหลัก โดยมีจุดประสงค์ในการลดกระดาษ กระบวนการทั้งหมดจะอยู่ในระบบ เน้นไปที่การส่ง email เพื่อการสื่อสาร

**กระบวนการทั่วไปเป็นดังนี้**

1. ผู้ขอเบิกด้วยเอกสาร Stock Request Order (ส่ง email)
2. ฝ่ายพัสดุเตรียมวัสดุและกำหนดวันรับวัสดุ (ส่ง email)
3. ผู้ขอมาที่ฝ่ายพัสดุเพื่อรับวัสดุในวันที่กำหนด ฝ่ายพัสดุจึงทำการตัดวัสดุออกจากคลัง (ส่ง email ยืนยันตามจำนวน)

![](img/sr_overview.png)

## User Roles

1. Stock Request User: ผู้ขอเบิกวัสดุ
2. Stock Request Manager: ฝ่ายพัสดุผู้ดูแลคลังวัสดุ

!!! Note
      - Stock Request User จะเห็นเฉพาะเอกสารของตัวเองเท่านั้น
      - Stock Request Manager จะเห็นเอกสารทั้งหมด

## Use Cases

 1. ขอเบิกวัสดุและเบิกได้ทั้งหมด
 2. ขอเบิกวัสดุแต่เบิกได้บางส่วน
 3. ขอเบิกวัสดุแต่ถูกยกเลิกโดยฝ่ายพัสดุ
 4. ขอเบิกวัสดุและยกเลิก
 5. การนับสินค้าและปรับยอดจำนวนวัสดุในคลัง
 6. รายงานที่เกี่ยวข้องกับคลังวัสดุ
 7. การใช้งานอื่นๆ
